Introduction

DamageControl is an extensible and platform neutral Continuous Integration system.
It builds on the ideas and experiences from CruiseControl, and aims to take it a step further by being:
  • Easier to install and configure
  • Easier to extend
  • Platform agnostic (I.e. work for non-java environments)
  • Event driven
  • Multiple notification schemes (IRC, RSS, HTML, mail, JIRA...)
  • Smarter SCM integration (automatic rollbacks of checkins causing build failures etc.)

Download

Currently you'll have to get it via CVS. It is a moving target, and nothing is stable or final.

cvs -d :pserver:anonymous@cvs.codehaus.org:/cvsroot/damagecontrol login

(just press the enter key)

cvs -z3 -d :pserver:anonymous@cvs.codehaus.org:/cvsroot/damagecontrol co damagecontrol

JIRA issue tracker

The requirements, progress and bugs are tracked in DamageControl JIRA.
