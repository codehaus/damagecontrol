Introduction

DamageControl is an extensible and platform neutral Continuous Integration system.

Architecture

DamageControl is a system built around the principles of an event driven architecture (EDA) and implemented in Ruby. At the core of the system is the Hub. The Hub is responsible for mediating messages between the various components in the system. The entire system is loosely coupled.

Download

Currently you'll have to get it via CVS. It is a moving target, and nothing is stable or final.

cvs -d :pserver:anonymous@cvs.codehaus.org:/cvsroot/damagecontrol login
(just press the enter key)

cvs -z3 -d :pserver:anonymous@cvs.codehaus.org:/cvsroot/damagecontrol co damagecontrol
